Project 1: Neural-Network-for-classification-from-scratch-with-NumPy

In this project, I aimed to build a simple neural network from scratch using only the NumPy library in Python. The main focus of this project was to deepen my understanding of the mathematical foundations and internal dynamics of neural networks. The network is trained to classify handwritten digits from the MNIST dataset and compared to an equivalent implementation in PyTorch.

Project 4 (group project): Reinforcement Learning - Performance Comparison of Q-learning and REINFORCE in Maze Solving
